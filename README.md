# Vibe Tracker 

Ever stumble upon a an interesting or unique location. Vibe tracker allows you to track these locations that you might want to revisit later.
Disclaimer: This takes a while to load due to it running on Heroku free tier.
https://vibetracker.herokuapp.com
## Running:
Must have npm installed.
```
$ npm install --prefix
$ npm install --prefix ./client
$ npm start ./client && node server.js
```
